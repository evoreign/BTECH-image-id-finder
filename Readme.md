
# BTECH image ID finder

A site to upload masterdoc, find which duplicated image id in which model


## Installation/running locally

Run the FE

```bash
  npm install btech
  cd btech
  npm run dev
```
Run the BE


```bash
  cd api
  npm start
```
    
## Tech Stack

**FE:** next js, Shad cn, TailwindCSS

**BE:** express js, Django

**DB:** MongoDB atlas

**User auth:** Clerk

**Server:** NPM
